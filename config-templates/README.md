# Configuration
- config.json
    - gossipsub_topics
        - video
            Set to something unique to you.
        - chat
            Set to something unique to you.
    - addresses
        - app_addr
            Streamer app video ingest address.
        - ffmpeg_addr
            Transcoding ingest address.
    - tracks
        Quality name.
        Level.
        Codec.
        Bandwidth.
    - segment_duration
        Number of seconds for each video segment.
